--- title: "Small form factor and edge computing with OpenShift and Kubernetes" abstract: "Research project exploring how OpenShift and Kubernetes can be optimized for small form factor and edge computing." cid: home sitemap: priority: 1.0 --- {{< blocks/section id="oceanNodes" >}} {{% blocks/feature image="flower" %}} [MicroShift]({{< relref "/docs/home" >}}), is a project exploring how OpenShift and Kubernetes can be optimized for small form factor and edge computing. MicroShift can be deployed as a systemd service either containerized or from an RPM. It is optimized for field-deployment and production-grade, ready to run Kubernetes and OpenShift workloads. #### Run MicroShift Anywhere MicroShift is open source and requires minimal configuration. With frugal use of resources, you can effortlessly move workloads out to the far edge with well established tools and patterns of Kubernetes and OpenShfit. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section id="video" background-image="earth_video_banner_homepage" >}}
